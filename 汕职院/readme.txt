请将此文件放入wamp环境下的www文件夹中即可运行。

css文件夹用于存放网站所需css文件
js文件夹用于存放网站所需js文件
img文件夹用于存放网站所需图片文件
conf文件夹用于存放各个网页的主要配置文件
lib文件夹用于存放网站所需公共方法文件
pri文件夹用于存放各个网页的主要结构文件
pub文件夹用于存放各个网页的头部和底部文件
sql文件夹用于存放网站所需数据库文件

index.php为网站首页
brief.php为学院简介页
leader.php为现任领导页
culture.php为校园文化页
campus.php为校园风景页
view.php为院系设置页
work.php为工作简报页
login.php为后台登录页(前端没有做连接接口)
form_check为检测用户登录账户密码页
check.php为检测后台账户密码错误而返回的页面
